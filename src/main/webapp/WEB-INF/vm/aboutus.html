我们实验室是一个热爱科研，热爱技术，热爱进步的集体组成的。“求是创新”是我们实验室不变的宗旨。

实验室目前主要从事分子模拟与计算、量化计算、锂电池材料研究等课题。

实验室现有高性能计算机56台，其中3台单台内存24G，其余皆为8核8G的高性能服务器。 系统主要为Linux，含有PBS队列管理系统，装有Gromacs、NAMD、LAMMPS、VASP、Gaussian、Ambertool、PWscf (Quantum Espresso)、Plumed、Materials Studio等主流分子模拟及量子化学计算软件。实验室还长期租用深圳超算计算机，适合计算高精度超大型体系。

实验室备有PVD、CVD薄膜生长仪器，电池制作设备，电池充放电仪，电化学工作站，共沸点测定仪等等，适合开展电子材料及制造技术研究，锂电池材料研究，太阳能电池研究， 热力学与热分析研究等工作，是一个理论、计算、实验多方面、多角度，相互支持、相互借鉴、相互促进的实验室。

实验室长期和美国北卡州立大学（North Carolina State University），德国马普学会研究所，瑞典皇家理工学院（KTH）、乌普萨拉大学，英国皇家理工大学，日本大阪大学等互动交流。在教8-218装有一套远程视频会议系统，可供开展国际学术Seminar和小型学术交流活动。

实验室的在读博士生，近半数都有一到两年出国交流的经历，毕业后的学生过半数都走上了大学教师或各种科研单位的岗位，在课题组博士的科研训练体系十分完备，在成为一位合格的化学科研工作者的同时，还有可能成为软硬件专家，Linux专家，编程专家，物理理论专家，教学专家，生物医学专家。这和我们浙大“KAQ，宽专交”的培养模式是一致的。宽就是研究范围广；专就是能够深入课题成为科研前沿，成果发表在专业顶级期刊上；交就是学科交叉互相借鉴，兼容并包。我们实验室有来自化学、材料、化工、药学等多个学科的同学，我们培养同学们的各项技能，使得在这里的博士经历成为人生中不可多得的学习经历。实验室在读的硕士也能够从事各类前沿课题的研究，比如枪械表面的疏水材料研究，太阳能电池的研究，锂电池材料的研究等等前沿课题。

我们十分欢迎热爱化学，热爱科学的本科生、研究生、博士后加入到我们的团队中。想进一步了解情况，请联系我们: qiwang@zju.edu.cn
